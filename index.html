Boîtes à lexique : progression A1 → B1

Une vue simple pour aider les élèves à voir comment le vocabulaire attendu évolue entre les niveaux A1, A2 et B1 selon les grandes thématiques.

A1 : mots du quotidien immédiat A2 : autonomie dans la vie courante B1 : expression plus précise et argumentée

Se présenter / Vie personnelle

A1

  • Nom, âge, nationalité, adresse
    • Dire son nom : Je m'appelle..., Moi, c'est..., Je suis..., Mon prénom, c'est..., Mon nom de famille, c'est...
    • Pour demander : Comment tu t'appelles ? / Comment vous vous appelez ? / Quel est ton prénom ? / Quel est ton nom de famille ?
    • Dire son âge : J'ai ... ans. Point de vigilance : on dit « J'ai 14 ans » et non « Je suis 14 ans ».
    • Pour demander l'âge : Quel âge as-tu ? / Tu as quel âge ? / Quel âge avez-vous ?
    • Dire sa nationalité / son origine : Je suis français / française ; Je viens de... ; Je suis de...
    • Pour demander : Tu es de quelle nationalité ? / Quelle est ta nationalité ? / D'où tu viens ? / Tu viens d'où ?
    • Dire où on habite / son adresse : J'habite à..., en..., au... ; Mon adresse, c'est... ; J'habite dans une maison / un appartement.
    • À travailler en priorité : à + ville (à Paris) ; en + pays féminin (en France) ; au + pays masculin (au Maroc).
  • Famille proche (mère, frère, etc.)
    • Présenter sa famille : Dans ma famille, il y a... ; J'ai... ; Je n'ai pas de... ; C'est mon père / ma mère / mon frère / ma sœur ; Voici mon père / ma mère.
    • Questions utiles : Tu as des frères et sœurs ? / Dans ta famille, il y a combien de personnes ? / Tu as une sœur ? / C'est qui ?
    • Donner des informations : Il/Elle s'appelle... ; Il/Elle a ... ans ; Il/Elle habite à... ; Il/Elle est...
    • Composition familiale : Nous sommes ... dans la famille ; J'habite avec... ; Mes parents sont... ; J'ai un petit frère / une grande sœur.
  • Goûts simples (j'aime / je n'aime pas)
    • Dire ce qu'on aime : J'aime... ; J'adore... ; J'aime bien... ; Mon activité préférée, c'est... ; Mon plat préféré, c'est...
    • Dire ce qu'on n'aime pas : Je n'aime pas... ; Je n'aime pas du tout... ; Je déteste...
    • Parler d'activités : J'aime + nom ; J'aime + infinitif ; Je préfère... ; Je joue à... ; Je fais...
    • Demander les goûts : Tu aimes... ? / Qu'est-ce que tu aimes ? / Quel est ton sport préféré ? / Quelle est ta musique préférée ? / Tu préfères... ou ... ?
  • Structures transversales indispensables
    • Se présenter : Bonjour, je m'appelle... ; J'ai ... ans ; Je suis... ; J'habite à... ; J'aime... ; Dans ma famille, il y a...
    • Poser des questions : Comment tu t'appelles ? ; Quel âge as-tu ? ; Tu habites où ? ; Tu as des frères et sœurs ? ; Tu aimes quoi ?
    • Réponses courtes : Oui ; Non ; Oui, j'aime ; Non, je n'aime pas ; Moi aussi ; Moi non plus.
    • Relier des idées : et ; mais ; aussi.
  • Mini production modèle A1 : « Bonjour, je m'appelle Nadia. J'ai 13 ans. Je suis marocaine. J'habite à Barcelone. Dans ma famille, il y a quatre personnes : mon père, ma mère, mon frère et moi. J'aime la musique et le basket. Je n'aime pas les maths. »
  • Version très synthétique pour une programmation : Je m'appelle... ; J'ai ... ans ; Je suis... ; Je viens de... ; J'habite à... ; J'ai... ; C'est mon / ma... ; Il/Elle s'appelle... ; Nous sommes ... dans la famille ; J'aime... ; J'adore... ; Je n'aime pas... ; Je déteste... ; Je préfère...

A2

  • études, profession, routines
  • traits de caractère fréquents
  • projets proches et expériences passées
  • Structures : « Je travaille comme vendeuse. » ; « D'habitude, je me lève à 7h. » ; « L'an dernier, j'ai fait un stage. »

B1

  • parcours personnel plus détaillé
  • qualités / défauts nuancés
  • objectifs, motivations, priorités
  • Structures : « Ce qui me motive, c'est... » ; « J'aimerais évoluer vers... » ; « Même si c'est difficile, je... »

École / Travail

A1

  • salle de classe, matières, horaires
  • consignes simples (écouter, écrire)
  • objets scolaires du quotidien
  • Structures : « Nous avons français à 10h. » ; « Ouvrez le cahier, s'il vous plaît. » ; « Il y a un stylo sur la table. »

A2

  • organisation de la journée
  • tâches et responsabilités simples
  • relations avec collègues / camarades
  • Structures : « Je dois préparer le dossier. » ; « On se retrouve à midi ? » ; « Je m'occupe de... »

B1

  • description d'un poste ou d'un projet
  • compétences et difficultés
  • opinion sur méthodes et résultats
  • Structures : « Je suis chargé de... » ; « J'ai du mal à..., mais je progresse. » ; « À mon avis, cette méthode est efficace. »

Maison / Ville / Déplacements

A1

  • pièces de la maison, meubles
  • lieux de la ville (gare, école, parc)
  • indications basiques (à gauche, tout droit)
  • Structures : « Ma chambre est petite. » ; « La boulangerie est en face de la gare. » ; « Tournez à droite. »

A2

  • services urbains (banque, poste, mairie)
  • types de logement et quartier
  • transports, horaires, itinéraires
  • Structures : « Je prends le bus pour aller au centre-ville. » ; « Le train part à 8h15. » ; « Mon quartier est assez calme. »

B1

  • avantages / inconvénients d'un lieu
  • comparaison ville-campagne
  • mobilité, environnement, aménagement
  • Structures : « Le principal avantage, c'est... » ; « La ville est plus animée que la campagne. » ; « Il faudrait améliorer les transports. »

Achats / Alimentation / Santé

A1

  • nourriture de base, repas
  • prix, quantités, monnaie
  • parties du corps, symptômes simples
  • Structures : « Je voudrais un kilo de pommes. » ; « Ça coûte combien ? » ; « J'ai mal à la tête. »

A2

  • faire des courses plus détaillées
  • goûts, habitudes alimentaires
  • conseils de santé courants
  • Structures : « Je mange souvent des légumes. » ; « Je préfère le thé au café. » ; « Vous devriez vous reposer. »

B1

  • consommation responsable
  • régimes, équilibre alimentaire
  • décrire un problème de santé avec précision
  • Structures : « J'essaie de réduire ma consommation de sucre. » ; « Depuis trois jours, j'ai une douleur au dos. » ; « Il vaut mieux éviter les produits ultra-transformés. »

Loisirs / Culture / Numérique

A1

  • sports et loisirs fréquents
  • jours, dates, activités du week-end
  • outils numériques de base
  • Structures : « Le samedi, je joue au foot. » ; « On se voit dimanche ? » ; « Je me connecte à l'application. »

A2

  • sorties, événements simples
  • médias (film, musique, réseaux)
  • actions en ligne courantes
  • Structures : « On va au cinéma ce soir ? » ; « J'ai regardé une série hier. » ; « J'ai partagé la photo sur le groupe. »

B1

  • critiquer un film, un livre, une chanson
  • exprimer préférences et justifications
  • parler des usages du numérique
  • Structures : « J'ai trouvé ce film captivant, car... » ; « Je préfère ce livre, parce que... » ; « Les réseaux sociaux peuvent être utiles, mais... »

Société / Environnement / Citoyenneté

A1

  • fêtes, règles de politesse
  • météo et saisons
  • gestes écologiques élémentaires
  • Structures : « Bonne fête ! » ; « Il fait beau aujourd'hui. » ; « J'éteins la lumière en sortant. »

A2

  • services publics de base
  • petits problèmes de la vie sociale
  • tri, recyclage, énergie au quotidien
  • Structures : « Je dois aller à la mairie. » ; « Il y a trop de bruit dans l'immeuble. » ; « Nous trions les déchets à la maison. »

B1

  • solidarité, participation, engagement
  • débattre de sujets de société simples
  • causes et conséquences environnementales
  • Structures : « Je me suis engagé dans une association. » ; « Je pense que cette mesure est utile. » ; « Si on consomme moins, on réduit les déchets. »
Vous pouvez adapter ces boîtes selon votre programme et ajouter des exemples de mots appris en classe.
